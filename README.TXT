#---------------------------GRAPH DRAWING  PROJECT-----------------------------
This project is meant to draw serval kinds of praphs(including triangle,circle,
rectangle,square,horizontal line,intersecting line). When executing, the 
program gives user a menu so that the user can choose a shape to draw. The mune
also contains a exit choice. 

Table of Content
1.Configuration
2.Installation
3.Operation
4.Manifest(list of files and short description of their roles)
5.License
6.Contact information

------------------------------------------------------------------------------

1.Configuration
---------------
Don't have to configurate any,but to make sure the computer is able to run .c 
file.

2.Installation
--------------
Copy link form git hub,then run make command, after that will be able to run 
the application. 

3.Operation
----------- 
Run make command should be able to see a graph.a file, then run  './graph.a' 
command then will be able to see a mune like below:
========= menu ============
(1). Draw Triangle
(2). Draw Circle
(3). Draw Rectangle
(4). Draw Square
(5). Draw Horizontal Line
(6). Draw Intersecting Line
(0). Exit
===========================
Input a integer number to make choice for wanted graph, input a integer number
according to the prompted text.
